# Mecatronica-UPB-Projects

### O colecție de proiecte de modelare 3D și ansambluri mecanice realizate în Autodesk Inventor și SolidWorks, incluzând piese de laborator și proiecte de robotică.

---

## 👨‍💻 Despre Mine
Sunt student în anul I la Facultatea de Inginerie Mecanică și Mecatronică (UPB), pasionat de tehnologie, modelare CAD și automatizări. Acest portofoliu reflectă evoluția mea în utilizarea software-urilor industriale de proiectare.

---

## 🛠️ Competențe Tehnice / Technical Skills
- **CAD:** SolidWorks, Autodesk Inventor, AutoCAD.
- **Hardware:** Proiecte de automatizare cu Arduino.
- **Software:** Python, C++, Java, HTML5/CSS.

### Angrenaj Planetar (Epicyclic Gearing)
*Software: Autodesk Inventor / SolidWorks*

O asamblare mecanică complexă ce utilizează roți dințate pentru obținerea unor rapoarte de transmitere ridicate într-un spațiu compact.

<div align="center">
  <img width="100%" alt="Angrenaj Planetar" src="https://github.com/user-attachments/assets/b3cb6581-e428-448d-97f8-e1fdf609353e" />
</div>

*Detalii proiect:*
* **Componente:** Roată solară, sateliți, port-satelit și coroană dințată.
* **Focus:** Verificarea interferențelor între dinți și simularea mișcării relative a componentelor.
* **Aplicație:** Mecanisme de reducere pentru motoare electrice și robotică.
